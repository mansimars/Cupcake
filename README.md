# Cupcake
An app that lets you choose and send order information of the cupcake you like

### Topics covered
* Kotlin Fundamentals
* Rows
* Columns
* Lazy Column
* Image
* Material Design
* Animations
* Navigations
* View Model 
* State & State Hoisting
* Adaptive Layout

### Software Stack
* Android Studio
* Jetpack Compose
* Kotlin
